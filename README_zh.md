本课程专为 C++ 编程的完全初学者设计。你将学习 C++ 的基础知识，包括数据类型、运算符、控制结构、函数、数组、字符串、类、动态内存、文件处理以及标准模板库（STL）。在课程结束时，你将能够编写和理解基本的 C++ 程序。
